# Project Structure
live_tracking_apps
  - apps
    -- frontend
    -- backend
  - package
    -- shared

## Backend Using Nestjs and socket io
using port 3001
Build
```sh
cd apps/backend
npm run build
```
Run
```sh
npm run start:dev
```

## Frontend using React Js and react-leaflet for Map

Run the following command:
using port 3000
```sh
cd apps/frontend
npm run dev
```
